## Project Description
This project demonstrates an Open Source Software Audit using Linux Bash scripts.  
It focuses on system information, package management, file auditing, log analysis, and open-source philosophy.

## Scripts Description

### Script 1: System Identity Report
Displays system information such as kernel version, user name, uptime, Linux distribution, and date.

### Script 2: FOSS Package Inspector
Checks if VLC is installed using package management commands and displays package details.

### Script 3: Disk and Permission Auditor
Analyzes important directories and displays their permissions and storage usage.

### Script 4: Log File Analyzer
Searches for keywords in log files and counts occurrences to help in debugging.

### Script 5: Open Source Manifesto Generator
Takes user input and generates a personalized open-source manifesto.

## How to Run the Scripts

### Step 1: Open Terminal
Navigate to the folder where scripts are saved.

### Step 2: Give Permission
chmod +x script1.sh

### Step 2: Run the command
./script1.sh

Repeat the same steps for other scripts by changing the file name.
